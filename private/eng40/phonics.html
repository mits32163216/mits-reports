📚 理論編：Day 2〜14（13回）─ 音の理屈と口の形を理解する（重森ちぐさ版56分）

Day 2表情筋を起こす ─ A と「吐き続ける息」

▶ 0:00〜4:52 を視聴

2つのコツ＝筋肉と息の使い方の説明→4つの筋肉（表情筋・唇・舌・お腹）→表情筋の鍛え方→A（ニコッと笑って口角を斜め上・顎だけ下げる）→日本語はプツプツ切れる息、英語は「出し続けて終わる」（急ブレーキではなくスーッと止まる車のイメージ）。

練習語：ask / match など（Aの単語5つ）。ニコッをキープしたまま顎だけ下げる。

Day 3曖昧母音（シュワ）と E

▶ 4:52〜8:25 を視聴

単語の母音はアクセントの1つだけしっかり発音し、残りは「曖昧な音」（お腹からゲップのように出す音）になる。pen・Japan で練習→E はスマイルのまま日本語の「え」を長めに伸ばす（日本語より圧倒的に長く）。

練習語：pen / Japan / egg / bed / ten など。2つ目の母音を曖昧にできたか。

Day 4I ＋ 唇の筋肉スタート B

▶ 8:25〜12:50 を視聴

I はスマイルして「潰れたイ」。舌の根元を奥歯の内側にギューッと押し出すイメージ。in / it / if / big の音が変わる→唇の筋肉に移り、B は唇を閉じるだけでなく巻き込んでお腹から息をポンと出す。

練習語：in / it / if / big / about / trouble / job など。ティッシュが動けば息が出ている。

Day 5P ＋ M（溜めて出す）

▶ 12:50〜16:56 を視聴

P は B と同じ口の形で音がない（無声音）。P に気を取られて表情筋を忘れないこと→M は唇を巻き込み、口を閉じたまま鼻から「ンー」。日本語の「ん」（難波・サンバの ん）と同じ音なので実はできる。溜めてから出すと綺麗になる（money）。

練習語：airplane / money / seven / from など。from の最後は唇を閉じたまま終わる。

Day 6W・Q・F（突き出して脱力）

▶ 16:56〜21:03 を視聴

W は唇をウッと突き出してから脱力して終わる。これも溜めてから出すと綺麗（what / was）→Q は W と同じ口の形で音を速く出す（question）→F は下唇に前歯を置き、お腹からの息をどこにも寄り道させずまっすぐ出す（無声音）。

練習語：what / was / wine / question / quiet など。question の -tion は曖昧母音で「シェン」。

Day 7F の応用 ＋ V（しびれる音）

▶ 21:03〜24:46 を視聴

family・famous・beautiful は日本語のカタカナにつられず、アクセント以外の母音を曖昧に。beautiful の ti は「D に近い音」→V は F と同じ口の形に音をつけ、唇がしびれる音が出れば正解。お腹から出す。

練習語：family / famous / beautiful / please / every など。every は「エヴリ」と2拍で。

Day 8舌の筋肉スタート ─ N を制する

▶ 24:46〜29:00 を視聴

「N を制したら発音が圧倒的に綺麗に変わる」回。N は口を閉じる「ん」ではなく、口を開けたまま舌を上につけて鼻から音を出す（「みんな」と言うときの ん）。banana は顎を2回動かさず、お腹から曖昧母音を出して舌だけ動かす。

練習語：nice / nine / banana / dictionary / station など。最後の N で口を閉じていないか鏡で確認。

Day 9L ＋ D（引き上げて止める）

▶ 29:00〜34:04 を視聴

L には2種類ある：語頭の L はしっかり、語尾の L は曖昧に発音（well はどちらでもOK）。日本語のラリルレロより深くお腹から→D は L と同じ舌の位置から下に弾く。語尾の D は「引き上げて止める」（grandchild / and）。カタカナの「ドクター」につられない。

練習語：well / hospital / doctor / grandchild / and など。and は「エン」から引き上げる。

Day 10T ＋ R（犬のうなり）

▶ 34:04〜38:20 を視聴

T は D と同じ口の形の無声音。語尾なら上げて止める、文が続くなら息を吐き出す（2種類の息の使い方）→R は唇を尖らせ、舌はどこにもつけない。巻きすぎ注意。犬が怒っている「ウー」の深い音をお腹から。W の音を少し入れて溜めてから出すと綺麗。

練習語：water / basketball / birthday / read / break など。舌は1回だけ動かす。

Day 11J・Y・X

▶ 38:20〜42:33 を視聴

J は舌先で上あごの奥をトンと叩く。顎を少し出す感じ（jacket / jam）→Y は舌の根元を上げて息を出し続け、上あごが少しかゆくなれば正解。「イヤー」ではなく摩擦の「イィ」から入る（year と ear の違いが分かるようになる）→X は「クス」。楽勝と思っても表情筋を忘れずに。

練習語：jacket / jam / yes / yellow / yesterday / year / box / six / taxi / textbook など。

Day 12お腹の音 ─ C・K・G・O・U

▶ 42:33〜46:56 を視聴

C と K は同じ「ク」（無声音）。GE が続くと J の音に変化（college）→G は同じ口の形の有声音。お腹が動いているか確認（good / go）→O はお腹からまっすぐ出す「ア」寄りの音（octopus は母音を入れずに）→U は「あっ、忘れ物した！」の短い「ア」（umbrella / bug）。

練習語：catch / college / kick / talk / good / go / octopus / onion / umbrella / bug など。

Day 13H・Z・S（かめはめ波と溜めるS）

▶ 46:56〜50:56 を視聴

H は息をかめはめ波のようにたくさん出す無声音（happy / have / has）→Z は歯と歯を合わせた間から濁った摩擦音。唇は脱力（力を入れると日本語の「ズ」になる）。jazz / prize は「吐きながら」→S は Z の無声版。溜めてから出すと綺麗。前が有声音の s は Z の音に変わる（dishes / is）。

練習語：happy / have / has / jazz / prize / say / socks / dishes / is など。

Day 14まとめ＋苦手音の総復習

▶ 50:56〜52:16 を視聴

まとめ：発音は知識ではなくスキル。ジムと同じで、真剣にやると表情筋もお腹も痛くなる（それが正解）。自転車のように考えなくてもできる「自動化」まで鏡を見て意識練習→視聴後、Day 2〜13 で一番苦手だった音を2つ選んで、その区間だけもう一度観てマネる。
