診断（4曲9軸データから見えた本質）

1. 中核テーマは「息の支え」

短音率・低音ノイズ・HNR・Jitter・Shimmer は全部同じ根っこ = 息の支え不足。Shimmer は 4 曲すべてで 13.5〜19.5%（体質的振幅揺らぎ）。バラードで大空 (HNR 10.22 / Jitter 1.90%) のように支えが利いた時のポテンシャルは高い。

2. 難易度で全指標が線形悪化

短音率: 57.6 → 57.2 → 67.0 → 78.4%（難曲ほど悪化）／Shimmer: 17.4 → 13.5 → 17.1 → 19.5%（一貫して高い・体質）／HNR: 6.56 → 10.22 → 5.83 → 3.75 dB（大空ピーク・努努底）。→ キー選びが 3ヶ月の成長速度を決める。

3. cent 変動は既に改善傾向

Wherever 27・努努 32.2。音程・耳のトレーニング（Phase D）の必要度は当初想定より低い。Phase D は日課化（Lesson 11）だけ残し、Phase B/C を厚くする。

4. 装飾テクニックは曲難易度で発動量が変わる

バラード（メロディ）: ビブラート6/しゃくり8/こぶし18/フォール6 = 余裕あり／ロック挑戦キー（Wherever）: 17/12/22/9 = 意識的技術投入／ロック限界キー（努努）: 2/4/7/3 = 音程を追うので精一杯。ビブラート幅の推移 183→154→125→173 cent = 支えが利く曲では洗練方向、限界曲では素朴に戻る。

5. 抑揚は 4 曲中 3 曲で「豊か」

声量コントロールの土台はある。抑揚系の教材優先度は低い。この強みは Phase B/C の「息の支え」改善で更に活きる。

6. リズムは全 4 曲で「不安定」

BPM 118〜138 全域で不安定。補講 S1（しらスタ手拍子）の重要度を格上げ、Phase A 併走で早めに取り入れる推奨。
