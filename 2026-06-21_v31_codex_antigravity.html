エンジン別 詳細

Claude Code Agent Teams（現行）
  • v3.1 が ~/.claude/teams/keiei-project/ で稼働中・25ノード
  • 全7機能対応・実績あり
  • 制約：experimental（CLAUDE_CODE_EXPERIMENTAL_AGENT_TEAMS）・team config が消えることがあるため起動時 ls 必須
Codex（OpenAI）— 実装済 / 未成熟
  • multi_agent v1 = stable / true
  • multi_agent_v2 で spawn_agent / send_message / close_agent / wait_agent 実装済（under development）
  • goals = stable / true・SQLite 永続のタスク管理
  • Claude Code → Codex 移行ツール /import（2026-06-09 リリース）
  • Mits 環境 v0.137・npm 最新 v0.141
Antigravity（Google）— 構造的不整合
  • peer 通信を設計思想として否定（人間が調整役の前提）
  • 再帰 spawn 禁止 → 参謀→リーダー→配下が物理的に作れない
  • team 永続構成も未実装
  • 独自強み：3層階層コンテキスト・Google Workspace MCP 標準搭載・Managed Agents API（Linux Sandbox クラウド実行）
